## 1. 문제 정의

### 1.1 현재 상황 (As-Is)
브랜드 마케팅 담당자들은 AI 인플루언서를 활용하고 싶지만, 어디서 어떻게 섭외해야 할지 막막합니다. 마치 수많은 포트폴리오 사이트를 일일이 방문해서 적합한 모델을 찾아야 하는 '바늘 찾기'와 같습니다. AI 모델 크리에이터들도 자신의 창작물을 어디에 홍보하고 수익화해야 할지 불분명합니다.

### 1.2 목표 상황 (To-Be)
브랜드는 마치 '배달앱'처럼 원하는 스타일과 콘셉트를 입력하면 AI가 자동으로 최적의 모델을 추천해주고, 클릭 몇 번으로 섭외를 완료할 수 있습니다. 크리에이터는 자신의 AI 모델을 등록하면 자동으로 적합한 프로젝트 제안을 받고, 안정적으로 수익을 창출할 수 있습니다.

### 1.3 왜 지금인가? (Why Now)
2024년 이후 AI 이미지 생성 기술이 상용화 수준에 도달했고, 실제로 AI 인플루언서를 활용한 마케팅 사례가 급증하고 있습니다. 하지만 이를 연결하는 플랫폼은 부재한 상황으로, '시장 선점' 기회가 있습니다.

### 킬러 질문 대응

**Q1: "네이버/카카오가 내일 똑같은 걸 만들면?"**
초기 3개월 내 100개의 고품질 AI 모델과 10개 이상의 성공 사례를 확보하여 '네트워크 효과'를 만듭니다. 크리에이터 커뮤니티를 통해 선순환 구조를 구축하고, AI 매칭 알고리즘의 정확도를 지속적으로 개선하여 기술적 해자를 만듭니다.

**Q2: "Rosebud AI처럼 수요가 부족해서 피벗해야 하면?"**
Rosebud AI는 셀프서비스 도구였지, 마켓플레이스가 아니었습니다. 우리는 크리에이터 생태계가 핵심이며, 수요 부족 시 Plan B(B2B 직접 영업)로 전환할 수 있습니다. (§9.4 피벗 옵션 참조)

**Q3: "가상 인플루언서에 대한 소비자 반발은?"**
2025년 브랜드 채택률 하락(86%→40%)은 소비자 직접 대면(B2C 마케팅)에서의 현상입니다. 우리의 타겟은 제품 촬영/콘텐츠 제작(B2B)이므로, 소비자가 가상 인물을 직접 팔로우할 필요가 없습니다. 모델이 아닌 이미지가 상품입니다.

**민지의 월요일 오전:**
출근하자마자 팀장에게 "이번 주 금요일까지 새 제품 SNS 콘텐츠 10개 필요해"라는 지시를 받습니다. 실제 모델 섭외하면 예산도 부족하고 시간도 촉박합니다. AI 모델 마켓플레이스에 접속해서 "캐주얼, 20대 여성, 밝은 분위기"라는 콘셉트를 입력하자 AI가 3명의 모델을 추천해줍니다. 마음에 드는 모델의 Standard 패키지(10장)를 선택하고 결제를 완료하니 5분 만에 섭외가 끝났습니다. 3일 후 고품질 이미지 10장이 도착했고, 금요일까지 여유 있게 캠페인을 준비할 수 있었습니다.

## 3. 사용자 스토리

### FEAT-0: 공통 흐름 (온보딩/로그인)
- [ ] 사용자로서, 이메일 또는 소셜 계정으로 간편하게 가입할 수 있어야 한다. 왜냐하면 복잡한 가입 절차는 이탈을 유발하기 때문이다.
- [ ] 사용자로서, 브랜드/크리에이터 역할을 선택할 수 있어야 한다. 왜냐하면 각 역할에 맞는 화면과 기능이 다르기 때문이다.

### FEAT-1: AI 모델 탐색/검색 (MVP)
- [ ] 브랜드 담당자로서, 스타일/용도/외형/인기도로 필터링하여 모델을 탐색할 수 있어야 한다. 왜냐하면 원하는 조건에 맞는 모델을 빠르게 찾기 위해서다.
- [ ] 브랜드 담당자로서, 모델 프로필에서 포트폴리오와 과거 작업물을 볼 수 있어야 한다. 왜냐하면 품질을 사전에 확인하고 신뢰를 형성하기 위해서다.
- [ ] 브랜드 담당자로서, 마음에 드는 모델을 찜할 수 있어야 한다. 왜냐하면 나중에 다시 찾아보기 편하게 하기 위해서다.

### FEAT-2: 섭외 요청/AI 자동 매칭 (MVP)
- [ ] 브랜드 담당자로서, 원하는 콘셉트와 스타일을 입력하면 AI가 최적의 모델 3~5명을 추천해주어야 한다. 왜냐하면 수백 개의 모델을 일일이 확인할 시간이 없기 때문이다.
- [ ] 브랜드 담당자로서, Basic/Standard/Premium 패키지 중 선택하여 즉시 결제할 수 있어야 한다. 왜냐하면 가격 협상 없이 빠르게 거래를 완료하기 위해서다.
- [ ] 브랜드 담당자로서, 포트원을 통해 안전하게 결제할 수 있어야 한다. 왜냐하면 신뢰할 수 있는 결제 시스템이 필요하기 때문이다.

### FEAT-3: 콘텐츠 제작/전달 (MVP)
- [ ] 크리에이터로서, 주문 목록에서 상태(접수/진행/완료)를 관리할 수 있어야 한다. 왜냐하면 효율적으로 작업을 추적하기 위해서다.
- [ ] 크리에이터로서, 완성된 콘텐츠를 업로드하면 브랜드가 즉시 다운로드할 수 있어야 한다. 왜냐하면 빠른 전달이 고객 만족도를 높이기 때문이다.
- [ ] 브랜드/크리에이터 모두, 주문별 채팅으로 실시간 소통할 수 있어야 한다. 왜냐하면 수정 요청이나 피드백을 신속하게 전달하기 위해서다.
- [ ] 크리에이터로서, 완료된 주문의 수익을 정산 받을 수 있어야 한다. 왜냐하면 지속 가능한 수익 창출이 플랫폼 참여의 핵심 동기이기 때문이다.

## 7. 해결 가설과 실험

### 7.1 가설
> 만약 브랜드에게 "콘셉트만 입력하면 AI가 최적의 모델 3~5개를 추천"하는 경험을 제공하면, 섭외 소요 시간이 기존 대비 80% 단축되고 만족도가 향상될 것이다. 왜냐하면 수백 개의 프로필을 일일이 확인하는 피로도가 사라지기 때문이다.

### 7.2 실험 루프

```
실험 → 관측 → 학습 → 다음 가설
```

| 단계 | 내용 |
|------|------|
| 실험 | AI 추천 vs 수동 검색 AB 테스트 (각 50명) |
| 관측 | 섭외 완료까지 소요 시간, 재검색 횟수, 만족도 설문(5점 척도) |
| 학습 | AI 추천 그룹이 평균 5분, 수동 그룹이 20분 소요 → 유의미한 차이 확인 |
| 다음 | 만족도 4점 이상이면 AI 추천 기본 제공, 3점 이하면 추천 알고리즘 개선 |
